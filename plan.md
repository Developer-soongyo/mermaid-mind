# MermaidFlow ‚Äî Project Plan

## Overview
A local web service that lets you:
1. **Render** Mermaid diagrams from raw markdown
2. **Generate** Mermaid diagrams from plain text descriptions using Claude AI
3. **Save & browse** diagram history with auto-generated names

---

## Tech Stack
| Layer | Choice | Reason |
|---|---|---|
| Runtime | Node.js | Lightweight, no extra install |
| Server | Express.js | Minimal, well-known |
| AI | Anthropic SDK (Claude Sonnet) | Diagram generation + auto-naming |
| DB | SQLite via `better-sqlite3` | Single file, zero config |
| Frontend | Vanilla HTML/CSS/JS | No build step, just open & run |
| Renderer | Mermaid.js (CDN) | Official library, client-side |

---

## Features

### 1. Render Panel
- Textarea for raw Mermaid markdown input
- Live preview rendered by Mermaid.js
- "Save" button with optional custom name (auto-named if left blank)

### 2. Generate Panel
- Textarea for plain English description
- "Generate" button ‚Üí calls Claude API ‚Üí returns Mermaid code
- Preview rendered immediately after generation
- Auto-saved to history with AI-generated name

### 3. History Sidebar
- Lists all saved diagrams (name + date)
- Click any item to load it into the render panel
- Delete individual entries
- Names are auto-generated by Claude if not provided

---

## API Endpoints

| Method | Path | Description |
|---|---|---|
| `POST` | `/api/generate` | Text ‚Üí Mermaid (Claude) + auto-save |
| `POST` | `/api/save` | Save raw Mermaid + optional name |
| `GET` | `/api/history` | List all saved diagrams |
| `GET` | `/api/history/:id` | Get single diagram |
| `DELETE` | `/api/history/:id` | Delete diagram |

---

## Database Schema
```sql
CREATE TABLE diagrams (
  id         INTEGER PRIMARY KEY AUTOINCREMENT,
  name       TEXT NOT NULL,         -- auto or user-provided
  source     TEXT,                  -- original text prompt (nullable)
  mermaid    TEXT NOT NULL,         -- the mermaid markdown
  created_at TEXT DEFAULT (datetime('now'))
);
```

## Setup Steps
```bash
# 1. Clone / create folder
mkdir mermaid-service && cd mermaid-service

# 2. Install dependencies
npm init -y
npm install express better-sqlite3 @anthropic-ai/sdk dotenv

# 3. Add .env
echo "ANTHROPIC_API_KEY=your_key_here\nPORT=3000" > .env

# 4. Run
node server.js
# ‚Üí http://localhost:3000
```
